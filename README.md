# Investor Reports

This repository contains a simple static website for investor report summaries.

## Structure
- `index.html`: Landing page listing available reports.
- `reports/amplifon-2025.html`: Detailed investor report for Amplifon FY 2025.

## Run locally
From the repository root:

```bash
python -m http.server 8000
```

Then open `http://127.0.0.1:8000` in your browser.
